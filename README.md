# Student Management System

A simple console-based **Student Management System** developed using **C++**. This project helps manage student records through a menu-driven interface.

## Features

- Add Student Record
- View Student Records
- Search Student by ID
- Update Student Information
- Delete Student Record

## Concepts Used

- C++
- Variables
- Functions
- Switch Statement
- Arrays
- For Loop
- While Loop
- Object-Oriented Programming (OOP)
- File Handling

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/Nandin12-bit/student-management-system.git
   ```

2. Open the project in VS Code or any C++ IDE.

3. Compile the program:
   ```bash
   g++ student_management.cpp -o student_management
   ```

4. Run the program:
   ```bash
   ./student_management
   ```
